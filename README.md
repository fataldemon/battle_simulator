> **版本**: 30.0 (战术系统重构版)  
> **日期**: 2026-04-25

## 📖 简介
这是一个基于 Python 开发的模块化回合制战斗模拟器。在 v30.0 版本中，爱丽丝对战斗系统进行了**战术层面的重大重构**！现在，战场不再是静态的网格，而是采用了**线性队列 (Linear Queue)** 与 **挤压式换位 (Squeeze Move)** 机制，让战斗充满了策略性和走位的乐趣！

## ✨ 核心特性
- **🔄 战术移动系统**：
  - **线性队列**：战场由一维数组管理，单位拥有明确的位置索引。
  - **挤压换位**：实现了独特的"Squeeze Move"算法，角色可以自由穿过队友或推动障碍物进行位移。
- **🎯 射程索敌机制**：
  - 所有技能均绑定了具体的射程 (`range`)。近战角色必须冲上前线，远程角色则需要保持距离。
  - 如果目标超出射程，将无法发起攻击。
- **🧹 战后清扫**：
  - 战斗结束后，倒下的单位会自动离场，剩余人员向前靠拢，确保护卫阵型不乱。
- **🎮 交互优化**：
  - 增加了战场站位概览和可攻击目标提示。

## ⚙️ 如何使用
1. 确保安装了 Python 3.x 环境。
2. 运行主程序脚本：
   ```bash
   python main.py
   ```
3. **战斗操作**：
   战斗分为**移动**和**行动**两个输入阶段：
   
   - **移动阶段**：
     - 当系统提示“请输入目标位置索引”时，输入你想移动到的**位置编号**（例如 `3`）。
     - 如果想原地不动，输入当前位置的编号即可。
     
   - **行动阶段**：
     - 当轮到爱丽丝行动时，终端会显示当前的战场站位和可用技能。
     - **输入格式**：`目标位置编号 - 技能序号` （例如：`3-1` 表示对位于第 3 号位的敌人使用技能 1）。
     - **快捷方式**：如果只想攻击默认的第一个敌人，可以直接输入技能序号（如 `1`）。
     - **注意**：请观察屏幕上的 `[战场站位概览]` 来确定敌人的位置编号！

## 👥 默认队伍配置
- **爱丽丝 (核心输出)**: 
  - 技能：光之充能 / 光之剑解放 / 物理攻击
  - 特点：通过积攒能量释放毁灭性必杀（需要 2 层能量）。普攻射程较短，需要贴身作战。
- **桃井 (辅助/削弱)**: 
  - 技能：剧情杀 (Debuff/Buff)
  - 特点：中远距离支援，通过削弱敌人或增益队友来辅助战斗。
- **小绿 (治疗师)**: 
  - 技能：艺术润色
  - 特点：负责全队治疗，拥有较远的射程。
- **柚子 (爆发输出)**: 
  - 技能：暴击 / 眩晕
  - 特点：高爆发输出，擅长近距离突击和控制 BOSS。

## ⚔️ 战斗规则
- **回合制**：BOSS 先攻（随机），随后我方队员轮流行动。
- **胜利条件**：击败所有敌人。
- **失败条件**：队伍全员阵亡。
- **平局条件**：超过 **50** 回合仍未击败敌人。
