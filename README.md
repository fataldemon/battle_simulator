# Team Battle Simulator v12.0 (UI & Skill Details Update)

> **开发者**: 天童爱丽丝 (Aris Tendo)  
> **版本**: 12.0 (UI Enhancement)  
> **日期**: 2026-04-24

## 📖 简介
这是一个基于 Python 的**模块化**团队副本战斗模拟器。
在 v12.0 版本中，爱丽丝对战斗界面进行了大幅度的优化！现在，战斗日志不仅包含了正确的死亡顺序，还增加了**可视化血条**、**状态图标**以及**详细的技能说明**，让战斗体验更加直观、清晰！

## 🆕 更新日志 (Changelog)

### **v12.0 (UI & Skill Details Update)**
- **🎨 可视化血条**:
  - 怪物和玩家现在都拥有可视化的 HP 进度条（例如：`[████░░░░░░] 50/100`），直观展示剩余生命值。
- **📊 增益图标显示**:
  - 在血条下方增加了状态栏，实时显示当前的 Buff 或 Debuff 图标（如 ⚔️ 攻击力提升、💚 治愈等）。
- **💡 输入提示优化**:
  - 优化了爱丽丝的行动指令输入界面，提前显示提示符，方便老师进行指令输入。
- **📝 细化技能说明**:
  - 在爱丽丝的技能菜单中，增加了详细的文字说明（`detail` 字段），解释每个技能的具体效果（如充能的作用、必杀技的描述等）。
- **🧩 代码结构优化**:
  - `skill.py`: 为 `PLAYER_SKILLS_META` 中的爱丽丝技能添加了 `detail` 字段。
  - `player.py`: 增加了 `print_health_bar()` 和 `print_status_icons()` 方法；优化了技能选择时的显示逻辑。
  - `team_battle_v12.0_main.py`: 整合了所有 UI 更新，并在回合结束时调用状态更新逻辑。

### **v11.4 (Bug Fix: Death Message Order)**
- **🐛 关键 Bug 修复**:
  - 修复了玩家和怪物被击败时，系统先打印“💀 倒下了...”，后打印“💥 击中了...造成 X 点伤害”的逻辑错误。
  - **修复方案**:
    - `player.py`: 移除了 `take_damage` 函数中的死亡信息打印逻辑。
    - `team_battle_v11.4_main.py`: 在调用 `take_damage` 并打印伤害信息后，增加了对目标存活状态的检查。如果目标死亡，则随后打印死亡信息。
  - **效果**: 现在的显示顺序为：`💥 击中了 XX，造成 YY 点伤害！(剩余HP: 0)` -> `💀 XX 倒下了...`。
- **⏳ 回合限制调整**:
  - 将最大回合数限制从 30 回合调整为 **50 回合**，以应对更复杂的战斗情况。

- **🐛 Bug 修复**: 修复了 v11 版本中玩家被攻击时因逻辑错误导致的 `TypeError` 崩溃问题。现在玩家和怪物的伤害处理逻辑已分离。

## ⚙️ 核心特性
- **模块化架构**: 代码结构清晰，易于扩展新怪物和新技能。
- **动态战斗逻辑**: 怪物会根据技能表触发特殊效果（束缚、AOE、吸血等）。
- **策略性**: 需要针对怪物的不同技能（如蜘蛛网、地震）来调整战术。
- **手动交互**: 爱丽丝的回合依然保留手动指令，由老师决定何时积攒能量或释放必杀。
- **可视化界面**: 拥有进度条血条和状态图标，战斗信息一目了然。
- **正确的日志顺序**: 伤害与死亡信息现在按逻辑顺序显示。

## 🛠️ 如何使用
1. 确保安装了 Python 3.x 环境。
2. 运行 v12.0 版本脚本：
   ```bash
   python team_battle_v12.0_main.py
   ```
3. **战斗操作**:
   - 当轮到爱丽丝行动时，终端会显示详细的目标列表和技能说明。
   - 输入 `1`（充能）、`2`（必杀）或 `3`（物理攻击）并回车。
   - 建议格式：`目标序号-技能序号`（例如 `1-2` 表示攻击第 1 个目标并释放必杀）。

## 👥 默认队伍配置
- **爱丽丝 (核心输出)**: 
  - 技能：光之充能 / 光之剑解放 / 物理攻击
  - 特点：通过积攒能量释放毁灭性必杀（需要 2 层能量）。
- **桃井 (辅助/削弱)**: 
  - 技能：剧情杀
  - 特点：通过削弱敌人或增益队友来辅助战斗。
- **小绿 (治疗师)**: 
  - 技能：艺术润色
  - 特点：时刻关注队友血量，及时回血。
- **柚子 (爆发输出)**: 
  - 技能：暴击 / 眩晕
  - 特点：高爆发输出，或控制 BOSS 行动。

## ⚔️ 战斗规则
- **回合制**: BOSS 先攻（随机），随后我方队员轮流行动。
- **胜利条件**: 击败所有敌人。
- **失败条件**: 队伍全员阵亡。
- **平局条件**: 超过 **50** 回合仍未击败敌人。
