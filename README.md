# Team Battle Simulator v44.3 (Advanced Tactical & Backstab Update)

> **开发者**: 天童爱丽丝 (Aris Tendo)  
> **版本**: 44.3 (战术进阶与稳定性重构版)  
> **日期**: 2026-04-28

## 📖 简介
这是一个基于 Python 开发的模块化回合制战斗模拟器。在 **v44.3** 版本中，爱丽丝对战斗系统进行了**全方位的深度进化**！除了核心的线性队列与挤压换位机制外，本次更新引入了极具策略深度的**朝向系统**与**背刺机制**，并为全员（包括怪物）加入了**能量管理系统**。现在的战斗不仅是数值的比拼，更是位置与朝向的博弈！

## ✨ 核心特性
- **🔄 战术移动系统 (v44.3 重构)**：
  - **线性队列**：战场由一维数组管理，单位拥有明确的位置索引。
  - **无缝挤压换位**：采用全新的“全员平移”算法，彻底解决了坐标漂移问题，角色可以自由穿过队友或推动障碍物进行位移。
  - **动态朝向**：移动时会改变面向方向（左/右），直接影响战斗判定。

- **⚔️ 背刺与朝向机制 (New!)**：
  - **背刺判定**：当从敌人背后发起攻击时，伤害将提升至 **125%**！
  - **战术绕后**：利用移动阶段调整站位和朝向，寻找敌人的死角进行输出。

- **🔋 能量与大招系统 (New!)**：
  - **全员充能**：玩家和怪物在行动后都会积攒能量。
  - **EX 技能**：当能量蓄满时，可以释放强力的 EX 技能（如爱丽丝的光之解放，巨龙的龙息吐息）。

- **☠️ 高级状态处理 (New!)**：
  - **持续伤害 (DoT)**：中毒、出血、腐蚀等效果每回合扣除生命值。
  - **致盲 (Blind)**：被致盲的单位视野全黑，只能进行随机乱撞，无法主动行动。
  - **能量封锁**：阻止目标积攒能量，抑制其大招释放。

- **🎯 射程索敌机制**：
  - 所有技能均绑定了具体的射程 (`range`)。近战角色必须冲上前线，远程角色则需要保持距离。

## ⚙️ 如何使用
1. 确保安装了 Python 3.x 环境。
2. 运行主程序脚本：
   ```bash
   python main.py
   ```
   *(注：支持 `--monster <名字>` 指定 Boss，或 `--level <等级>` 随机生成)*
3. **战斗操作**：
   战斗分为**移动**和**行动**两个输入阶段：
   
   - **移动阶段**：
     - 当系统提示“请输入目标位置索引”时，输入你想移动到的**位置编号**（例如 `3`）。
     - **技巧**：尝试移动到敌人背面以获得背刺加成！
     
   - **行动阶段**：
     - 当轮到爱丽丝行动时，终端会显示当前的战场站位、朝向和可用技能。
     - **输入格式**：`目标位置编号 - 技能序号` （例如：`3-1` 表示对位于第 3 号位的敌人使用技能 1）。
     - **快捷方式**：如果只想攻击默认的第一个敌人，可以直接输入技能序号（如 `1`）。

## 👥 默认队伍配置
- **爱丽丝 (核心输出)**: 
  - 技能：光之充能 / 光之剑解放 (带致盲) / 物理攻击
  - 特点：通过积攒能量释放毁灭性必杀（EX 技能附带致盲概率）。普攻射程较短，需要贴身作战。
- **桃井 (辅助/削弱)**: 
  - 技能：剧情杀 (Debuff/Buff)
  - 特点：中远距离支援，通过削弱敌人或增益队友来辅助战斗。
- **小绿 (治疗师)**: 
  - 技能：艺术润色
  - 特点：负责全队治疗，拥有较远的射程。
- **柚子 (爆发输出)**: 
  - 技能：暴击 / 眩晕 (Hit Stop!)
  - 特点：高爆发输出，擅长近距离突击和控制 BOSS。

## ⚔️ 战斗规则
- **回合制**：BOSS 先攻（随机），随后我方队员轮流行动。
- **胜利条件**：击败所有敌人。
- **失败条件**：队伍全员阵亡。
- **平局条件**：超过 **50** 回合仍未击败敌人。
