# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [Unreleased]

## [v30.0] - 2026-04-25
### Added
- **战术系统重构**：引入线性队列 (Linear Queue) 概念，战场由一维数组管理。
- **挤压式换位 (Squeeze Move)**：实现 `utils.py` 中的挤压算法，支持单位自由穿插与推挤。
- **基于射程的索敌**：所有技能均绑定 `range` 属性，实现了 `_find_valid_targets` 逻辑。
- **回合流程拆分**：战斗循环拆分为严格的【移动阶段】→【行动阶段】。
- **战后清扫**：阵亡单位自动离场，剩余人员向前靠拢。
### Changed
- **输入格式升级**：指令格式从“目标序号”变更为“位置编号”（如 `3-1`）。
- **UI 提示优化**：新增战场站位概览与可攻击目标列表显示。
- **技能平衡性**：重新规划了全量技能的射程梯度（近战 1-3，中程 4-6，远程 7-10，全屏 99）。
- **代码结构**：重构了 `main.py`, `player.py`, `monster.py`, `skill.py` 以适应新机制。

## [v21] - 2026-04-25

### Fixed
- **桃井 Debuff 显示修复**：将桃井的 Debuff 持续时间从 1 回合延长至 3 回合，确保状态栏图标在战斗中可见。
- **怪物状态栏修复**：修改了 `Monster.print_status()` 方法，使怪物血条旁能正确显示状态图标（如 📉）。
- **即时反馈机制**：在 `main.py` 的玩家行动结束后增加了“即时状态刷新”步骤，Buff/Debuff 无需等待下一回合即可立即显示。
- **台词重复 Bug**：修复了柚子大招喊话重复打印两次的问题。

### Updated
- **技能名称更新**：爱丽丝普攻名称修正为“光之剑，出鞘吧”。
- **充能台词更新**：爱丽丝充能技能名称修正为“光啊！赐予我力量！”。
- **代码优化**：改进了 `process_player_actions` 中对剧情 Buff 的处理逻辑。

## [16.0] - 2026-04-25
### Fixed
- **修复队友血量显示 Bug**：解决了 v15 版本中我方角色（爱丽丝、桃井等）的血量进度条不显示的严重问题。
- **控制台输出优化**：为所有关键打印语句添加了 `flush=True` 参数，确保血条和状态信息实时刷新，不会被战斗日志覆盖。

## [15.3] - 2026-04-25
### Added
- **怪物死亡台词**：当怪物被击败倒地时，系统会打印其专属的 `death_msg` 台词，让战斗更有临场感！
### Changed
- **移除怪物去重限制**：在自定义副本模式中，移除了 `available_monsters.remove` 逻辑。现在允许生成重复的同种怪物，方便灵活凑齐等级和挑战！
### Fixed
- **等级过滤修复**：修复了等级筛选逻辑，防止在低等级副本中意外刷出等级过高（如 Lv.99）的强力怪物。

## [15.2] - 2026-04-24
### Added
- **命令行参数支持**：增加了 `argparse` 模块，允许通过命令行参数自定义副本。
  - `--level`：指定怪物总等级，系统会随机生成满足等级的怪物群。
  - `--monster`：指定怪物列表（支持名字模糊匹配或索引），用于直接挑战特定 Boss。
- **自定义副本模式**：新增了 `init_game_custom` 函数，实现了非随机模式的副本生成逻辑。

## [15.1] - 2026-04-24
### Fixed
- 修复了优香大魔王（优香大魔王）在战斗中随机发动“未知技能”的 Bug。
- 在 `skill.py` 的技能注册表中补全了缺失的 `critical_hit` 技能定义。

## [15.0] - 2026-04-24
### Changed
- **玩家 UI**：将爱丽丝的能量槽更新为复古方块风格 `[█░]`。
  - 0 能量：`[░░]`
  - 1 能量：`[█░]`
  - 支持无限长度（超过 10 个格子时会显示省略号）。
- **技能信息**：为爱丽丝的大招增加了动态倍率显示。
  - 示例：`(倍率 x11.82)`
- **逻辑**：修复了能量计算逻辑，确保至少显示 2 个格子。

## [14.1] - 2026-04-24
